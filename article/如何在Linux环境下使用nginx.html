# 基本操作

安装nginx

brew install nginx

Nginx配置文件目录：

/usr/local/etc/nginx/nginx.conf

启动

nginx

重启Nginx

nginx -s reload

关闭

# 查询nginx对应端口号
ps -ef|grep nginx

kill -QUIT [port]

# Nginx解决方案-反向代理

# 反向代理

客户端请求服务器，服务器接收转发给其他服务器获得内容，再传给客户端

# 正向代理

位于客户端和原始服务器之间的服务器，为了从原始服务器取得内容，客户端向代理发送一个请求并指定目标，然后代理向原始服务器转交请求并将获得的内容返回给客户端

# 疑难

  1. 为什么配置了静态资源，但是提示404 Not Found
server {
	listen 8080;
	server_name localhost;
	location / {
		root /Users/$user/server; # 指向项目目录
		#index index.html index.htm;
	}
	location /static/ {
		alias /Users/$user/server/static; # 静态资源路径
		autoindex on; # 是否开启目录索引
	}
}

在指定了静态资源的alias之后，必须保证location /也指向同一个项目目录，否则就会产生错误
